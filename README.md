# BD Shop — учебный интернет-магазин для портфолио

**Описание:**
Учебный проект интернет-магазина внутриигровых предметов MMORPG Black Desert Online.  
Проект полностью тестовый, не связан с Pearl Abyss, все игровые материалы принадлежат правообладателям.

**Технологии:**
- Python, Django, Django REST Framework
- PostgreSQL
- JWT (djangorestframework-simplejwt)
- React (минимальный фронтенд)
- Docker (опционально)

**Функционал:**
- Каталог товаров и категорий
- Корзина с добавлением/удалением товаров
- Оформление заказов с фиксацией цен
- Админ-панель для управления товарами, заказами и пользователями
- Минимальный frontend для просмотра каталога, корзины и заказов
